סכוּמילים (קוֹגרמה)

בעברית ישנן 22 אותיות, ואלו יכולות לשמש לשיטת ספירה בבסיס 22 בייקטיבי (bijective) כלהלן:
א = 1   ב = 2  ... י = 10  כ = 11 ... ש = 21  ת = 22 = ø (כמו בשעון)

כעת נוכל לפרק משפט לשני משפטים חדשים, כך שסכום כל זוג אותיות מקביליות במשפטים החדשים הוא אות המקבילה במשפט המקורי. למשל: 

עושות אהבה ולא מלחמה
===== ==== === =====
הסתכם שורת חדש ושימש
+++++ ++++ +++ +++++
כמשפט בשדה רחב זמרתו

נא לשים לב שבניגוד לחידת אותיות ומספרים (alphametics או cryptarithms), כאן הערכים המספריים של האותיות קבועים מראש.

Cogram - in Hebrew there are 22 letters, which can be used as a numerical system in bijective base 22. Using modular arithmetic, we can "factor" a sentence into two new sentences, so that each pair of parallel letters in the new sentences sums up to the parallel letter in the original sentence. Note that in contrast to cryptarithms or alphametics, here the numerical values of the letters are predetermined.



סכוּמילים פלינדרומי (קוֹדרום או סכוּמוֹפֹה)

באופן דומה, נוכל לפרק משפט פלינדרומי (להלן של ערן הדס) לשני משפטים פלינדרומיים חדשים: 

רק פושטק עלוב בולע קטשופ קר
== ===== ==== ==== ===== ==
ער הדסים דעתי יתעד מיסדה רע
++ +++++ ++++ ++++ +++++ ++
דש לבושו לצון נוצל ושובל שד

Codrome - similarly, we can factor a palindromic sentence (as the above by Eran Hadas) into two new palindromic sentences.



סכוּמילים פלינדרומי מוצלב (קוֹדרום כיאסטי)

לחלופין, נוכל לפרק משפט פלינדרומי לזוג משפטים אנדרומיים שיחד יוצרים פלינדרום אחד גדול: 

רק פושטק עלוב בולע קטשופ קר
== ===== ==== ==== ===== ==
ער הדסים לצון יתעד ושובל שד
++ +++++ ++++ ++++ +++++ ++
דש לבושו דעתי נוצל מיסדה רע

אם במשפט המקורי יש מספר אי־זוגי של אותיות, אז האות האמצעית חייבת להיות בעלת ערך זוגי (ב', ד', ...).

Chiastic codrome - alternatively, We can factor a palindromic sentence into a pair of anadromic sentences that together form one big palindrome. If the original sentence has an odd number of letters, then the middle letter must be of even value.
