רֶסֶן הוא כתב עת מקוון ליצירה אילוצית וחישובית בעברית. כתב העת שם לעצמו למטרה לתת במה והקשר רחב, תרבותי, היסטורי וקוסמופוליטי, לכותבים הפועלים בתחומים הללו, כמו גם להרחבה וגיוון של קהילת הקוראים והחוקרים בתחום. מתפרסמות בו יצירות מקור, תרגומים וביצועים מחדש של יצירות מן העולם וכן מאמרים ומסות בנושאי יצירה אילוצית וחישובית.

עיצוב לוגו: מיתר מורן.

Logo design: Meytar Moran.

כתב העת רֶסֶן הופק בסיוע מועצת הפיס לתרבות ולאומנות לשנת 2023 על סך 20,000 ש"ח, ולשנת 2024 על סך 29,000 ש"ח. ובסיוע מענק מיוחד מטעם קרן יהושע רבינוביץ לאומנויות תל־אביב לשנת 2023 על סך 8,000 ש"ח, לשנת 2024 על סך 10,000 ש"ח, ולשנת 2025 על סך 6,000 ש"ח.

רֶסֶן הוא אתר רשת שנוצר בעבודת יד, רץ בצד־לקוח בלבד ללא תלות במשאבים חיצוניים, ומוגש כקוד פתוח. האתר אינו אוסף נתונים כלשהם מהמבקרים. האתר מתארח ב־GitHub Pages, שאוסף נתוני מבקרים לצורכי אבטחה. בגישה לאתר זה דרך שם מתחם שונה מ־github.io (בפרט resen.co.il), שירותי הפניית אתרים ופרוקסי (בפרט Cloudflare) אוספים נתוני מבקרים, באופן המזערי הניתן להגדרה. איננו משתמשים בנתונים אלו או משתפיהם. האתר מכיל קישורים לאתרים חיצוניים בעלי מדיניות פרטיות שונה.

Resen is a handmade, self-contained client-side only website, served as open source. The website does not collect any data from visitors. The website is hosted on GitHub Pages, which collects visitor data for security purposes. When accessing this website via a domain name other than github.io (namely resen.co.il), URL redirection and proxy services (namely Cloudflare) collect visitor data, with the minimal configurable settings. We do not use nor share this data. The website contains links to external websites with different privacy policies.
